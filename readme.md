## 2.制作アプリの説明（40文字程度）
- 前回のくじ引きをベースにキャラ立ち。誰がやるかズバッと決めてくれるアプリ
## 3.工夫した点・こだわった点
- 前回できなかった複数人当選ができる
- オープニングの文字表示
- 画面遷移時に入力した値をパラメータで引き継ぐこと
## 4.次回トライしたこと（または機能）
- レスポンシブデザインが完全ではないのでもっと調整したい
- エフェクトをもっと使いたい
## 5.備考（感想、シェアしたいこと等なんでも）
- 画像は自力＋Geminiで作成。Figma、Cancvaなども活用。Geminiが変な画像を返してくるのでもっとうまくコントロールできるようになりたい。
- 課題がうまくはまればシリーズ化してみたい
- 例）どこでもいい子ちゃん　お店選びをズバッと決めてくれる
- 例）いつでもいい代ちゃん　スケジュールをズバッと決めてくれる
